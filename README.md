# HackTracker - Futuristic Hackathon Tracker

HackTracker is a modern, futuristic web application designed to help developers and tech enthusiasts discover, track, and participate in the most innovative hackathons worldwide. It features an interactive calendar, detailed event views, and a clean, intuitive user interface.

## Features

- **Interactive Calendar**: A sleek, responsive calendar to view upcoming hackathon registration dates.
- **Detailed Event Views**: Click on a date to see detailed information about the hackathons, including prize money and participant numbers.
- **Multi-Event Support**: The calendar visually indicates days with multiple hackathons.
- **Featured Hackathons**: A dedicated section to highlight top hackathons.
- **Floating Navigation**: A stylish, right-aligned navigation pane for easy access to different sections.
- **Newsletter Subscription**: A section for users to subscribe to a weekly newsletter for hackathon updates.
- **Optimized for Performance**: Integrated with Vercel Speed Insights for performance monitoring.
- **SEO Friendly**: Enhanced metadata for better search engine visibility and social sharing.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [npm](https://www.npmjs.com/) (or your preferred package manager like pnpm or yarn)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/hackathon-tracker.git
    cd hackathon-tracker
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
